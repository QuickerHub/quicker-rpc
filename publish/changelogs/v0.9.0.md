v0.9.0

## CLI

- `qkrpc` 目标框架升级为 **.NET 10**（`global.json` 固定 SDK）
- 动作项目 `info.json` 改为 protobuf 模型与 JSON 映射；`action extract` 从 `action get` 元数据写入 `info.json`
- `action extract --min-lines` 默认改为 4；表达式步骤自动外置为 `.eval.cs`
- 发布脚本与 CI 继续以 `version.json` 为唯一版本源

## 安装

- GitHub Release 产出 `qkrpc-win-x64-setup.exe`、便携 zip 与 `quicker-agent-win-x64-setup.exe`（Tauri NSIS）
- QuickerAgent 下载页支持 Bitiful 版本标记与从 `version.json` 推导文件名

## 插件 / Quicker

- 加固表达式占位符解析（eval 相关）
- 发布时允许并发 qkrpc 管道会话

## agent-gui

- 工作区资源管理器与 `.quicker/actions` 磁盘编辑工作流（extract / sync / patch）
- 动作项目 `info.json` 与 Quicker 元数据同步；workspace 文件工具增强
- 长对话自动上下文压缩；批量工具审批
- 修复 Tauri 下 `window.confirm` 与 dialog 插件 2.7 不兼容导致的运行时错误
- 无头编辑文档合并为单一 Agent Skill（`action-steps` / `action-variables` 等主题）
