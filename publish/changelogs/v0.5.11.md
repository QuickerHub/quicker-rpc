v0.5.11

CLI
- **新增** `qkrpc fa search`：Font Awesome 图标搜索（默认压缩为 `Light_*` / `Brands_*`；`--expand` 返回全部样式）
- **新增** `qkrpc action set-metadata`：无头更新动作标题、说明、图标（`--icon` 支持 `fa:{enumName}[:#color]`）
- `action patch` / `subprogram patch` 可在同一保存中附带顶层 `title`、`description`、`icon`
- 图标 spec 保存前校验（非法格式或未知枚举返回 `errorMessage`）
- `inputParams` 未知键不阻止保存，在响应 **`warnings[]`** 中给出合法键与相近建议（`step-runner get` 提示）
- `guide` 专题：`overview`、`action-icons`；Agent 文档压缩，细节下沉到专题
- `build.ps1` 兼容 Windows PowerShell 5.1

安装
- 与 v0.5.0 相同：`qkrpc-win-x64-setup.exe` + zip 便携包；Release 与 CI 共用本 changelog

插件 / Quicker
- Font Awesome 目录搜索 RPC；动作元数据无头写入
- patch 保存前 `inputParams` 键名警告收集
- 集成测试：元数据/图标、FA 搜索
