CLI

安装

插件 / Quicker

- 版本号 **0.13.3.0**（与 `quicker.rpc` 依赖包一致，第三段正式发布）

QuickerAgent

- **多工作区**：侧栏分「工作区 / 对话」两层，可按目录切换工作区与对应对话列表
- **对话持久化**：SQLite 改用 Drizzle ORM；迁移前自动备份；保存/import 守卫防止误清空历史
- **Workbench**：工作区侧栏 Git 改动列表、资源管理器视图切换
- **稳定性**：修复 `workspaces` 字段缺失导致白屏；chunk 热更新失败自动刷新；友好错误页
- **历史恢复**：「从老版本恢复」合并导入时强制写入消息 blob；SQLite 空时回退 localStorage 分片
- **UI**：工作区标题行与侧栏折叠按钮不再重叠
