v0.12.4

CLI

- **`qkrpc mcp install`** 扩展：支持 VS Code Copilot、Windsurf、Cline；`--all` 一键写入各宿主用户级配置；`--project` 同时生成 `.cursor/.vscode/.mcp.json` 与 Claude Code `.mcp.json`
- VS Code 使用 `servers` + `type: stdio` 格式；其余宿主使用 `mcpServers`
- 新增 **`qkrpc wait`**：轮询等待 QuickerRpc 插件就绪（MCP `qkrpc_wait` 同步）
- 集成文档：[docs/agent-mcp-integration.md](../agent-mcp-integration.md)

安装

- GitHub Release：`qkrpc-win-x64-setup.exe` / 便携 zip（CI 构建）
- QuickerAgent：`quicker-agent-win-x64-setup.exe`（GitHub Release + Bitiful 镜像）

插件 / Quicker

- 版本号 **0.12.4.0**（与 `quicker.rpc` 依赖包一致，第三段正式发布）
- Agent 工具按意图拆分（query / run / create / manage 等），减少 mega-tool 误用
- `data.json` 步骤与变量默认值 wire 压缩，降低 token 占用
- 程序结构 lint 与 agent-gui 工具链扩展
- QuickerAgent：@-mention 支持子程序/图标/引导；步骤参数编辑器对齐桌面 Quicker；Tauri 启动器与更新流程加固
- 分享动作更新说明与本 Release 一致
