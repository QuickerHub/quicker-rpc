v0.12.0

CLI

- 版本号 **0.12.0.0**（与 `quicker.rpc` 依赖包一致）
- **`qkrpc action run --wait`**：步骤失败时返回 `errorMessage` / `stopFlag`，不再误报 `ok: true`
- serve / JSON 输出同步暴露动作运行错误字段
- 动作 trace feed 路由与 SSE 核心拆分，侧栏调试流更稳定

安装

- GitHub Release：`qkrpc-win-x64-setup.exe`、便携 zip、**QuickerAgent** Tauri 安装包
- 用户安装：https://github.com/QuickerHub/quicker-rpc/releases/latest/download/qkrpc-win-x64-setup.exe

agent-gui（QuickerAgent）

- **启动性能**：工作区侧栏默认关闭，打开后才加载资源树与文档目录（消除初始卡顿）
- **聊天性能**：消息窗口虚拟化、冷轮次折叠占位、流式阶段纯文本渲染、`localStorage` 延迟写入
- **Thought 块**：活动批次内嵌 reasoning 流式展开、结束后自动折叠
- 工具批次流式时省略过早步骤；仅挂载当前活动聊天标签
- 动作 trace 面板与 feed 客户端重构

插件 / Quicker

- **`PluginAssemblyResolve`**：包目录 `AssemblyResolve`，修复 `_packages\quicker.rpc` 下 `Microsoft.Extensions*` 等依赖加载失败
- Costura 继续嵌入 `Microsoft.Extensions*`（不改为仅卫星 DLL）
- `XActionTraceRunService` 与 RPC 回调小幅整理
- `quicker.rpc` 依赖包同步至 **0.12.0.0**
- 分享动作更新说明与本 Release 一致
