v0.11.13

CLI

- 版本号 0.11.13.1（与 `quicker.rpc` 依赖包一致；revision 热更新走依赖包编辑替换）
- **`qkrpc action run --trace`**：XAction 结构化 trace 输出（步骤/输入/输出/变量/耗时）
- **`qkrpc launcher resolve`**：统一解析设置页、动作、子程序候选（Agent 选路）
- **`qkrpc settings resolve`**：设置意图 dry-run（关键词 / preset / key → 打开目标）
- serve：`/actions/trace/stream` SSE；MCP / HTTP 动作 trace 与 launcher 工具

agent-gui（QuickerAgent）

- **动作 trace 面板**：聊天 / 工作区侧栏 / tool-test 同步展示 RPC trace 时间线
- **Launcher Agent**：自然语言打开设置、解析动作与子程序（tool-test + 聊天工具）
- **Quicker 设置意图**：预设用例与 dev 校验路由；设置页目录与直达链接扩充
- **退出确认 overlay**、侧栏 trace 切换；工具输出 / ActionLink 卡片与 prompt 菜单优化
- **tool-test**：launcher / settings-intent / action-trace 独立面板与回归用例

插件 / Quicker

- **XAction trace**：插件侧 `XActionTraceRunService` + RPC 回调流式推送调试事件
- **LauncherResolveService**：设置 / 动作 / 子程序统一打分排序
- **QuickerSettingsUiService**：设置页 catalog、resolve、headless apply 增强
- **`QuickerRpc_Run`**：QExpr 启动路径修复（`launch_code` + `registration` load/type，避免父级 eval 解析失败）
