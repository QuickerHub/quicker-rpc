v0.12.5

CLI

- 新增 **`qkrpc agent setup`**：用户级一键安装 MCP、skills、Cursor rules 与 `~/.qkrpc/agent-setup.json` 版本清单；`qkrpc mcp install` 保留为别名
- **`qkrpc agent setup --upgrade`**：CLI 升级后仅刷新 skills/rules/Claude 指引，**不覆盖**已有 MCP 配置
- **`qkrpc agent setup --check`**：对比 manifest 与当前 CLI 版本，提示重跑 setup
- Skill 家族：`qkrpc`、`quicker-authoring`、`quicker-sync`、`quicker-run`；Cursor rule `qkrpc.mdc`；Claude `CLAUDE.md` 自动 merge
- **`GET /openapi.json`** 与 **`qkrpc serve openapi`**：HTTP 自建 Agent 的 OpenAPI 3.1 导出
- npm 薄包装 **`@quickerhub/qkrpc-mcp`**（spawn `qkrpc mcp`，skills 仍走 `agent setup`）
- 分发说明：[docs/agent-skill-distribution.md](../agent-skill-distribution.md)

安装

- GitHub Release：`qkrpc-win-x64-setup.exe` / 便携 zip（CI 构建）
- QuickerAgent：`quicker-agent-{semver}-x64-setup.exe`（GitHub Release + Bitiful 镜像）
- 升级后建议：`qkrpc agent setup --upgrade`

插件 / Quicker

- 版本号 **0.12.5.0**（与 `quicker.rpc` 依赖包一致，第三段正式发布）
- 安装包捆绑 `skills/`、`agent-rules/` 供 `agent setup` 离线复制
- QuickerAgent：trace flush 定时器类型与浏览器 `setTimeout` 对齐
- 分享动作更新说明与本 Release 一致
