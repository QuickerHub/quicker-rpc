v0.9.18

## CLI

- **step-runner search**：基于 curated 关键词表（`step-runner-agent-keywords.json`）的字段加权检索，结果含 `snippet`；文档与 `guide` 已同步
- **action project**：新增 `export` / `import` / `apply` / `validate`，支持 `.quicker` 项目目录与 file ref 编译
- **action publish**：分享/更新 getquicker 分享动作（`action update` 为别名）
- **action move**：将动作移动到指定动作页
- **action list / search**：支持 `uses:` / `uses-only:` 公共子程序引用查找；无 query 时默认按最后编辑时间排序
- **action edit-var**：无头修改变量默认值（不打开设计器）
- **profile create / reorder**：创建全局面板页、调整全局页签顺序
- **process ensure-ceacore**：创建 CeaCore Run 虚拟进程（可选 `--move-actions` 归集专用动作）
- **serve**：扩展 `/v1/invoke` 操作面（与 agent-gui 工具对齐）

## 插件

- 动作搜索/列表 scope 解析与引用扫描（`ActionSubProgramCallScanner`）
- 全局动作页创建、虚拟进程创建、CeaCore 宿主、动作分享上传等 RPC 服务
- 无头变量编辑服务重构（`HeadlessVariableEditService`）

## QuickerAgent（agent-gui）

- 工作区资源管理器：动作项目树、同步栏、编辑器分栏
- 工具调用展示与审批流程优化；中断工具调用修复
- 应用版本检查与设置页；Tauri 侧进程/job 管理改进
- `qkrpc serve` 生命周期管理（启动/复用/退出）

## 安装

- GitHub Release：`qkrpc-win-x64-setup.exe`（CI 构建）及便携 zip
- Quicker 依赖包 `quicker.rpc` 0.9.18.0
