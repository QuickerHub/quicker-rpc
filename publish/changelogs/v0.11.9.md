v0.11.9

CLI

- 版本号 0.11.9.0（与 `quicker.rpc` 依赖包一致）
- **动作 list/search**：JSON `filter` / `sort` / `fields` 投影；`ActionSummaryFieldCatalog` 字段白名单
- **`qkrpc settings links` / `open --preset`**：预设直达链接（快捷键、回收站、外观等），一步打开 Quicker 设置页
- **Bootstrap**：`QuickerRpcBootstrapPolicy` 避免 Quicker 未运行时重复弹插件启动
- MCP / serve：`qkrpc_action_query` / `qkrpc_action_manage` 工具面拆分（兼容旧 action 名）

安装

- GitHub Release：`qkrpc-win-x64-setup.exe`、便携 zip、**QuickerAgent** Tauri 安装包
- 用户安装：https://github.com/QuickerHub/quicker-rpc/releases/latest/download/qkrpc-win-x64-setup.exe

agent-gui（QuickerAgent）

- **自动更新 UX**：检查/下载改为右下角静默 toast，不再全屏阻塞；安装失败可重试；splash 结束后再检查更新
- **Agent 工具拆分**：`qkrpc_action_query` / `qkrpc_action` / `qkrpc_action_manage`；子程序 query/manage 同理；旧工具名自动映射
- **嵌入式浏览器**：WebView bounds 与滚轮转发修复
- **Quicker 设置**：`quicker_settings` 支持 `links` 与 `open --preset`
- 启动 splash 超时兜底；语音设置文案等小修

插件 / Quicker

- `quicker.rpc` 依赖包同步至 0.11.9
