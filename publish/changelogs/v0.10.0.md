v0.10.0

CLI

- 动作工程（action project）支持内嵌子程序：`subprograms/` 目录 compile/export 往返，CLI 与 serve 均可用
- Form Spec 完整链路：compile / export / decompile，动作步骤中的表单定义可外置为 `.form.json` 并还原
- `action project` serve 操作扩展；step-runner 目录 serve 缓存，减少重复 RPC
- agent-gui 工作区编辑器：资源树、步骤/变量/程序编辑、表单定义可视化编辑、动态步骤摘要
- 多 endpoint LLM 配置与有序 fallback；CI Release 嵌入 `BUNDLED_LLM_CONFIG`

安装

- GitHub Release 提供 `qkrpc-win-x64-setup.exe`、版本 zip 便携包及 QuickerAgent Tauri 安装包
- 用户安装：https://github.com/QuickerHub/quicker-rpc/releases/latest/download/qkrpc-win-x64-setup.exe

插件 / Quicker

- `HeadlessActionProgramService` 与 AgentModel 支持动作内嵌子程序编译
- `quicker.rpc` 依赖包同步至 0.10.0
- 分享动作更新说明与本 Release 一致
