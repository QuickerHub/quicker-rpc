v0.14.5

QuickerAgent

- **Agent 执行引导**：系统提示增加 Agent / Launcher 执行循环（先定目标、先查再改、批量读、失败重试、验证后收尾）
- **运行时上下文**：每轮对话注入模式、cwd、模型与已启用工具列表，便于模型对齐当前环境
- **上下文压缩**：触发阈值提高到约 90% 窗口用量（对齐 Cursor 等主流 Agent），短会话保留完整工具历史
- **回复语言**：从对话推断用户语言并注入 system prompt
- **动作范围**：@-pin 标签作为可选提示，不再从工具结果推断 scope
- **动作设计器**：嵌入设计器、step-module 引用与 wire 校验；setup 验证 JSON 与 chat hydration 改进
- **Authoring**：12 个子 skill 晋升、step-comments skill、学习循环脚本与 benchmark 更新

CLI / 插件

- **FA 图标搜索**：倒排 token 索引，加快 `fa search` 与 MCP 检索
- **Mock 验证**：Agent mock verify loop、设计器工具与 authoring benchmark 基础设施
- **quicker.rpc 0.14.5.0**：与 QuickerAgent 0.14.5 同步发布
- `QuickerRpc_Run` 子程序 `version` 同步为 **0.14.5.0**
