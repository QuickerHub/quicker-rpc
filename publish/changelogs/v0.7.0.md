- 作者文档改为生成管线维护，CI 校验文档与源码一致

## 插件

- 与 CLI serve 模式及 agent API 配套的服务端改动

- 新增 Next.js + Tauri 桌面 Agent：对接完整 qkrpc 工具链与 serve 调度
