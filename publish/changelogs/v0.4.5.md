v0.4.5

CLI
- `action run --json` 输出包含 `debug`、`wait`，与运行选项一致
- `action run --debug` 帮助文案明确为「调试运行并打开 Quicker 步骤调试器」
- `help --json` 中 `action run` 文档补充 `--debug` 选项

安装
- 发布流程：Release 说明与 CI 共用仓库内 `publish/changelogs/vX.Y.Z.md`（须先 commit 再打 tag）

插件 / Quicker
- Agent 动作编写指南：无专用模块时 **优先 `sys:csscript`（C#）**，`sys:runScript` 仅用于极短系统命令或用户已有脚本（`implementation-fallback` 等主题已更新）
