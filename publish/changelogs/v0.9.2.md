v0.9.2

## QuickerAgent

- **动作作用域**：@ 引用单个动作时，系统提示注入 Action scope；`workspace_action_*` 等工具拒绝写入其它动作 id，减少误改
- **工具失败详情**：失败的工具调用可展开查看请求/响应 JSON，便于排查
- **文件编辑卡片**：行级 diff 展示、浮动菜单布局与样式优化
- **工作区资源管理器**：删除动作项目确认流程改进；探索器偏好（展开状态等）持久化
- **会话标题**：自动标题生成与 API 路由调整，更稳定地跟随对话内容
- **聊天 UI**：工具批次分组、摘要标题与全局样式（工具卡片、侧边栏等）调整

## CLI / 插件

- 本版主要为 QuickerAgent 体验更新；CLI 与插件与 0.9.1 行为一致，随 Release 同步版本号
