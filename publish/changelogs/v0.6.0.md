v0.6.0

CLI
- **新增** `action export` / `action import`：导出/导入 `.quicker` 动作工程（`info.json` + `data.json`，保留文件引用为外部文件）
- **新增** `subprogram export` / `subprogram import`：子程序工程导出/导入（同上）
- **新增** `action float`：将本地动作显示为悬浮按钮
- `step-runner get` 支持控制字段联动：按 `--control-field` 过滤可见 `inputParams`（与 Quicker 设计器一致）
- `step-runner search` 增强关键词解析（`|` OR、`*` 通配、`sys:` 前缀等）
- `guide` 新增专题 `action-project-files`（工程目录布局与 file ref 编译规则）

安装
- 与 v0.5.x 相同：`qkrpc-win-x64-setup.exe` + zip 便携包；Release 与 CI 共用本 changelog

插件 / Quicker
- **QuickerRpc Monitor** 监控窗：qkrpc 虚拟动作页 + 全库最近编辑 XAction 双标签；RPC 连断、无头写入、窗口激活时防抖刷新
- 插件加载时可 `openMonitor: true` 自动打开监控窗；`Launcher.ShowMonitor()` / `ToggleMonitor()`
- `FloatActionAsync` RPC；无头写入后通知监控窗刷新
- StepRunner 目录映射控制字段与 `ValidForValues` / `VisibleExpression` 可见性规则
- XAction 文件引用导出/编译（`XActionFileRefExporter` / `Compiler`）
- 移除独立 `AssemblyLoader`（改由 Quicker 宿主加载）
- 集成/单元测试：工程 file ref、StepRunner 可见性、ActionProgramContent
