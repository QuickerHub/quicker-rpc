# QuickerAgent — 懂 Quicker 的 AI 副驾

> 开发文档：[agent-gui/README.md](../agent-gui/README.md) · 文档索引：[README.md](README.md) · 路线图：[ROADMAP.md](ROADMAP.md)

**QuickerAgent** 是专为 [Quicker](https://getquicker.net/) 打造的桌面 AI 助手。它直接连上你本机的 Quicker，帮你**找动作、跑动作、改动作、整动作页**——内置动作编辑器与步骤知识库，写流程不靠猜参数，日常操作不必翻遍面板。

不是通用聊天工具，而是长在 Quicker 旁边的效率层。

---

## 一句话

**找得到、跑得动、改得快——`CommandOrControl+Shift+Space`（Ctrl+Shift+Space）秒开，复杂编排交给主窗口慢慢聊。**

---

## 它解决什么问题？

动作越积越多，真正费时间的往往不是「会不会用 Quicker」，而是这几件事：

| 卡点 | QuickerAgent |
|------|--------------|
| 动作记不清名字，搜半天 | 按用途描述就能定位并执行 |
| 改步骤要反复开设计器、查模块参数 | 内置编辑器 + 懂步骤模块，按规范补全参数 |
| 新想法想马上试 | 直接运行或逐步调试，侧栏看每步输出 |
| 从零搭一条流程太累 | 多轮协作：创建、编排、整理、发布一条龙 |
| 只想顺手改个设置或标题 | 启动器浮窗，办完即走 |

QuickerAgent **不替代** Quicker，而是把「找—跑—改—发」这条链缩短。

---

## 核心能力

### 操控 Quicker，不用记细节

描述你想做的事就行，不必背动作名、不必记设置路径：

- 运行某个用途的动作
- 改标题、描述、图标
- 打开 Quicker 里某一页设置
- 新建动作并加上 HTTP、脚本等步骤

它会搜索动作库、执行或让你点选，回复也是普通人能看懂的说明，不堆术语。

### 动作：从想法到分享

| 阶段 | 能帮你做什么 |
|------|-------------|
| 发现 | 关键词、用途、子程序引用——多种方式找到目标动作 |
| 运行 | 一键执行；需要排查时逐步调试 |
| 编辑 | 应用内动作设计器，字段与步骤体验贴近 Quicker |
| 编排 | 添加步骤、变量、子程序调用；先查模块定义再落笔 |
| 整理 | 移动动作、管理动作页布局 |
| 分享 | 发布到 getquicker 动作库 |

**懂 Quicker 才是重点**：写步骤前会查模块 schema 和图标库，而不是凭空编造参数——这是它和通用 AI 的本质差别。

### 全局启动器

按 **`CommandOrControl+Shift+Space`**（Ctrl+Shift+Space，可改）唤起轻量浮窗：

- 打开设置、运行动作、改元数据——**马上要做**的事，不进主窗口
- 关窗即结束，不污染长对话
- 支持 `@` 直接点名某个动作

主窗口负责**复杂编排**；启动器负责**眼前这一下**。

### 串联本机工作

需要时还能在工作目录里跑命令、读写文件、打开网页查资料，把 Quicker 自动化和日常事务接在一起。

## 谁适合用？

- **Quicker 老用户**：动作库大了，需要一个「懂你家底」的检索与执行入口
- **动作作者**：少点鼠标，把精力放在逻辑上；调试、改步骤、整理动作页更省力
- **想快速试自动化的人**：从草稿到可运行动作，周期更短

## 怎么开始？

### 下载

[下载 Windows 版（x64）](https://github.com/QuickerHub/quicker-rpc/releases/latest/download/quicker-agent-win-x64-setup.exe)

### 使用前

1. 电脑已安装 **[Quicker](https://getquicker.net/)** 并保持运行
2. 在 Quicker 中安装并加载 **QuickerAgent** 插件动作  
   [从 getquicker 获取 →](https://getquicker.net/Sharedaction?code=aa5917ad-1256-4c73-7022-08debe3efcbe)
3. 目前支持 **Windows x64**

### 安装后

1. 在 Quicker 里运行 **QuickerAgent** 动作，或从开始菜单打开应用
2. 正式版内置对话模型，安装好即可使用；也可在设置里切换自己的模型

## 常见问题

**没有 Quicker 能用吗？**  
不能。QuickerAgent 是 Quicker 的搭档，需要本机已安装 Quicker 并完成插件加载。

**我的动作数据安全吗？**  
与 Quicker 的交互在本机完成。你发给 AI 的聊天内容，由所选模型服务商处理——和日常用大模型聊天一样，可按需在设置里选用自己的接口。

**能完全离线吗？**  
智能对话需要联网访问大模型；与 Quicker 的配合始终在本机。

**应用里的编辑器和 Quicker 设计器一样吗？**  
日常编辑都能覆盖；个别高级场景仍可一键跳转到 Quicker 桌面设计器继续改。
