# sys:getExplorerPath

> **分类**：文件与目录 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [getexplorerpath](https://getquicker.net/KC/Help/Doc/getexplorerpath)

**用途**：Get Explorer current folder path

用于获取当前资源管理器窗口的路径，或跳转到指定路径。

[image]

# 获取路径

获取资源管理器窗口所打开的目录路径。

[image]

输出参数

【当前窗口路径】当前具有焦点的资源管理器窗口路径。

【所有打开的路径】所有资源管理器窗口打开的路径的列表。

【最近访问的路径】（在使用其它软件时）获取最近打开的资源管理器窗口的路径。示例动作：[切换](https://getquicker.net/Sharedaction?code=b7a369da-ec8e-4d0f-764b-08d950c2afd2)

# 设置路径

用于跳转前台资源管理器窗口到指定的目录。

[image]

输入参数

【路径】需要跳转到的目标目录路径。
