# sys:comment

> **分类**：常用基础 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [comment](https://getquicker.net/KC/Help/Doc/comment)

**用途**：Comment-only step for documentation

# 概述

用于在步骤列表中插入一段注释文字，给予自己或其他人必要的信息。如：

- 后续一组步骤的功能用途；
- 如何根据自身需求定制动作；
- 方便以后修改动作的信息等...
