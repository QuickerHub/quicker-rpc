# sys:selectFolder

> **分类**：文件与目录 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [selectfolder](https://getquicker.net/KC/Help/Doc/selectfolder)

**用途**：Pick folder via dialog

选择一个文件夹，获取其完整路径。

[image]

# 参数

【提示文字】显示在选择窗口标题栏的文字。

【显示已打开的文件夹】

启用：显示已经在Windows资源管理器中打开的文件夹，方便直接选择。

[image]

不启用：直接显示选择文件夹窗口：

[image]

【取消后停止】用户取消选择后，停止当前动作。

# 输出

【是否成功】是否选取了一个文件夹。

【路径】选取的文件夹的完整路径。
