# sys:getClipboardFiles

> **来源**：step JSON 示例 · **官方**：[getclipboardfiles](https://getquicker.net/KC/Help/Doc/getclipboardfiles)

**用途**：读取剪贴板中的文件路径列表。

## 示例

### 读取剪贴板文件

```json
{
  "stepRunnerKey": "sys:getClipboardFiles",
  "outputParams": {
    "isSuccess": "成功",
    "output": "文件列表"
  }
}
```
