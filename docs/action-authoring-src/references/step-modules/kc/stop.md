# sys:stop

> **分类**：程序流控制 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [stop](https://getquicker.net/KC/Help/Doc/stop)

**用途**：Stop action or return from subprogram

停止动作或从子程序中返回。类似于编程语言中的**return**语句。

[image: image.png]

# 操作类型

**默认：**​

- 如果在主程序中：停止当前动作。
- 如果在启用了“忽略错误”选项的步骤组中：只跳过后面的步骤，不停止动作，从此步骤组下面的模块开始执行。
- 如果在子程序中：结束当前子程序，返回到主程序中。

**​停止动作：**​

- 停止动作，即便是在子程序中使用，也会停止整个动作。

# 参数

**【标记为出错】**

用于中止子程序时，标记为子程序出错。此时如果在“[运行子程序](https://www.yuque.com/quicker/help/subprogram)”模块中选择了“失败后停止”选项，则会停止动作。

**【返回值】**

当动作被其他动作调用时，也可用于向其他动作返回执行结果信息。

[image: image.png]

当从子程序返回时，如果选择了“标记为出错”选项，则用以传递错误消息。
