# B03 列表循环 + 条件 break — 2026-06-13

## 预期

列表 `["a","b","target","d"]` 遍历，在 `item=="target"`（count=2）时 `break`，`found=target`，不处理 `d`。

## 实际

| 阶段 | 输出 |
|------|------|
| evalexpression | `items` = string[4] |
| each 迭代 0–1 | `a`,`b` → simpleIf False，跳过 |
| each 迭代 2 | `target` → `found=target` → break |
| showText | `text` = `target` |

动作 id：`3950e311-1d20-40bb-91cc-cb3d7ce348ec`（已删除）

## 卡点

**`sys:if` 无 Else 分支**：条件 False 时 NullReference（「未将对象引用设置到对象的实例」）。

## 应对

单分支条件改用 **`sys:simpleIf`**；`each` 子步骤写在 **`ifSteps`**。

## 是否需改 skill

已产出 `loop-control` pattern + draft `quicker-authoring-loop-control`。
