# sys:restoreActiveWindow

> **分类**：系统与窗口 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [restoreactivewindow](https://getquicker.net/KC/Help/Doc/restoreactivewindow)

**用途**：Restore previous foreground window focus

讨论

                    相关动作

活动窗口是指当前具有输入焦点的窗口。

本模块用于将活动窗口恢复为运行动作之前的活动窗口，用以解决目标窗口丢失焦点的情况。（如使用了用户选择模块或在动作中切换到了别的软件等情况
