# sys:textCounter

> **分类**：文本处理 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [textcounter](https://getquicker.net/KC/Help/Doc/textcounter)

**用途**：Count lines, characters, words in text

讨论

                    相关动作

统计文本的字符数量、行数等信息。

[image: image.png]

# 参数

## 输入

【文本】要统计数据的原始文本内容。

## 输出

【行数】文本的行数。

【字符数】总字符数量，中文文字算作1个字符。

【可见字符数】去除空白字符之后的可见字符数量。
