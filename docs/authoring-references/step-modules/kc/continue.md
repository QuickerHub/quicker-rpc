# sys:continue

> **分类**：程序流控制 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [continue](https://getquicker.net/KC/Help/Doc/continue)

**用途**：Skip to next loop iteration

在循环中，跳过后续步骤，开始下一次循环。

类似于编程语言中的**continue**语句。

与“跳出循环”的主要区别为：

- 跳出循环：中止所在循环模块的执行，不再循环下一次。
- 跳过后续步骤：不中止循环，而是跳过本次循环的后续步骤直接开始下一次循环。
