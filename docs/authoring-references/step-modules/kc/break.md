# sys:break

> **分类**：程序流控制 · **来源**：KC 官方文档（`npm run docs:modules:gen`）· [break](https://getquicker.net/KC/Help/Doc/break)

**用途**：Break out of each/repeat loop

在循环模块（如“[每个](https://www.yuque.com/quicker/help/each)”，“[重复](https://www.yuque.com/quicker/help/repeat)”等）中，**跳过后面的步骤 **并 **结束循环**。

类似于编程语言中的**break**语句。

如：总共需要循环100次，在第10次时跳出循环，则后面的90次不会再执行。
