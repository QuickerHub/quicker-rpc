当前版本 __RELEASE_TAG__ · 查看更新说明

下载 Windows 版 (x64)

安装包来自 GitHub Releases 官方渠道 · 也可使用短链 /download

/download 仅下载 qkrpc CLI 使用说明 GitHub 仓库

使用前请确认

  • 已安装并运行 Quicker，且已加载 QuickerRpc 插件（与 Release 同版本的插件包）。
  • 正式安装包（Release）已内置 LLM 配置，一般无需手动创建 llm-config.json；从源码开发时才需要。
  • 默认工作目录为 Documents\QuickerAgent（首次启动自动创建）。
  • 仅支持 Windows x64；暂无 macOS / Linux 安装包。

安装后

  • 运行 QuickerAgent，确保 Quicker 与插件处于运行状态。
  • 安装包内包含 Next.js UI、便携 Node 与 qkrpc，可自动连接本机 qkrpc serve（若已启动）。
  • 遇到问题请查看 agent-gui 文档 或 Release 说明。
